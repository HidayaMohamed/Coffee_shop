# COFFEE SHOP

#### Date, 2025/11/24

#### By *Hidaya Mohamed Mohamud*

## Description
This project is a Python application designed to practice Object-Oriented Programming, focusing on classes, relationships, properties, validation, and class methods.

It models a simple coffee ordering system using three main classes:

Customer

Coffee

Order

The goal is to show how objects interact through relationships.

## Installation
1.Create virtual environment (pipenv)
pipenv --python 3.8.13
pipenv install pytest requests
pipenv shell

Afterwards clone this repository and cd into it.
